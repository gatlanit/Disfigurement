I made this game just about a year ago for a Java course I took at my school. I'm uploading now after a year of not touching this project just to have a resting place for this game as well as a place to show off its code for future reference. 

You are free to fork this game and make it better as long as you follow license terms.

## Installation
- You need the following:
  - Java
  - OpenGL 4.6 capable GPU (recommended)
  
> Unfortunetly I do not remember much about this project since it has been over a year since I made this game so your milage may very on forking it
---
- Download Binary from releases [or Itch.io](https://gatlanit.itch.io/disfigurement)
> Binary is located in ```lwjgl3/build/lib``` if you forked it
